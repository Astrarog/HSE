size_t &  seed,

)

Функция генерации данных и для подсчёта времени выполнения функции evaluator.

Аргументы
[in]evaluator– функция, принимающая массив данных, время работы которой необходимо измерить
[in]lengs– длины массивов, которые надо генерировать и над которыми надо производить замер времени.
[in]seed– начальное значение для генератора случайных чисел
[in]count_samples– количество образцов по которым будет браться среднее значение
[in]only_averages– флаг, говорящий, что нужно выводить только среднее временя выполнения или же все проведённые эксперименты
Возвращает
среднее время работы функции evaluator

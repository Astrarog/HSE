Проект в рамках лабораторной работы № 5 по дисциплине "Методы программирования"
Выполнил студент СКБ-171 Астраханцев Роман

Цель проекта в использовании паттерна "Мост" для реализации объекта «множество», которое представляется различными структурами данных в зависимости от числа элементов (при этом «множество» должно уметь изменять внутреннюю структуру в зависимости от фактического количества объектов).
Этот паттерн полезен когда необходимо разделять абстракцию и реализацию на две отдельные иерархии классов так, что их можно изменять независимо друг от друга.
Например для создания разного типа объектов имеющих одинаковых интерфейс, который в свою очередь является архитектурозависемым.
В рамках данного проекта реализация объекта «множество» была построена по принципу перестроения внетрунней структуры с хеш-таблиц (setOnHashes) на деревья (setOnTrees) при большом количестве элементов для избежания коллизий.
